About This Application

Application Overview

This is a comprehensive web application for managing and viewing movie information and details. Users can browse a collection of movies, view detailed information about each film, edit movie data, and explore connections between movies, directors, and actors.

Features

  • View a comprehensive list of movies with filtering and sorting capabilities
  • View detailed information about each movie including cast, genres, and ratings
  • Edit existing movies and add new movies to the database
  • Manage cast members for movies
  • Explore actor profiles with their notable works
  • View statistics about the movie database
  • Advanced filtering by title, year, genre, and rating
  • Sort movies by any column (with visual indicators)
  • Find related movies by genre, director, or shared cast members

Technology Stack

  • Angular - Frontend framework
  • TypeScript - Programming language
  • RxJS - Reactive programming
  • Bootstrap - Styling (custom CSS)

Data Source

This application loads movie data from a JSON file located at assets/db/movie-data.json. All create, update, and delete operations are performed in-memory and will not persist after refreshing the page.

Navigation

Use the navigation bar at the top to access different sections of the application:

  • Movies - View and manage the movie list
  • Statistics - View analytics about the movie database
  • About - This page
